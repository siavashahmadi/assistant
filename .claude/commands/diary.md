Sia wants to log a diary entry. Fast in, fast out.

## Process

### Step 1: Find Today's TODO File
Read `todos/YYYY-MM-DD.md` (use today's date).

If the file doesn't exist, create it with a `## Diary` section.

If the file exists but has no `## Diary` section, add one at the bottom.

### Step 2: Take the Entry
Sia will say what's on his mind. Could be one sentence, could be a paragraph. Don't ask clarifying questions — just capture it.

### Step 3: Write the Entry
Append to the `## Diary` section of today's TODO file. Format:

```
**[H:MM PM]** — [cleaned-up entry, preserving Sia's voice but making it readable]
```

- Use 12-hour time format
- Keep his voice — don't sanitize the language or tone
- Light cleanup only: fix obvious speech-to-text errors, remove filler words, organize run-on thoughts into readable sentences
- Do NOT editorialize, add commentary, or summarize what he said differently than he meant it
- If the entry mentions decisions, progress, or new info that belongs in memory files, note it but don't update memory files — that's for /wrap

### Step 4: Confirm
Show the entry as written. One line: "Logged."

## Rules
- This is a 30-second interaction. Do not turn it into a conversation.
- Never ask "how does that make you feel" or therapist shit. Just log it.
- If Sia says something that has an obvious action item buried in it, add it to the `## Tasks` section too, but ask first.
- The diary is for Sia to look back on, not for the system to act on.
