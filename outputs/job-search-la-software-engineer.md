**1. Affirm SE II** — Best experience-to-requirement ratio. He's overqualified on experience, strong Python/AWS/Kubernetes match, fintech payment experience from T-Mobile is directly relevant, and the remote role spans $142K-$210K. Apply first.

**2. AMP Cloud Engineer** — His IoT data pipeline story at Joulea (10M+ daily points, RabbitMQ, fault-tolerant, hot/cold storage) is almost a word-for-word match for this job description. Meaningful work (climate tech), good culture fit for a small technical team.

### Resume bullet emphasis for these roles:

**Lead with the pipeline story** — nearly every LA role values data infrastructure:
> "Designed high-throughput IoT data ingestion pipeline processing 10M+ data points daily through RabbitMQ with fault-tolerant architecture"

**AI/LLM bullets are differentiators** — most SE candidates can't say this:
> "Built generative AI data labeling pipeline using OpenAI API... increasing labeling speed by 67% while maintaining 90% accuracy"
> "Developed LLM-powered chatbot using LangChain + RAG achieving 85% accuracy"

**Scale bullets close the deal at fintech/consumer companies:**
> "Developed 8+ production-ready microservices... processing 5,000+ daily payment transactions with 99.9% uptime" (Affirm, BlackLine)
> "Built enterprise Angular application for prepaid.t-mobile.com serving 20M+ customers" (Snap, Whatnot)

**Cost/performance numbers signal architectural judgment:**
> "Reduced infrastructure costs by 57% while maintaining sub-second query performance" (Dropbox, AMP)
